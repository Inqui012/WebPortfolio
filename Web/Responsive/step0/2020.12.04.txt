
* 웹의 반응형은 하나의 CSS로 화면 크기에 따라서 pc와 모바일을 동시에 구현하는것. 적응형은 pc와 모바일의 CSS, html이 따로 존재함.
    반응형은 모든 화면크기에 맞춰가면서 확인을 해야하기 때문에 손이 많이감.

* CSS 에서 사용하는 @import 에서 세미콜론(;) 앞에 용도를 지정 가능. 
    기본으로는 screen 으로 설정되어 있음.

* @import 의 용도지정은 CSS 내부에서 @media(미디어쿼리) 로 연결해줄 수 있다.
    ex> @media 용도이름 and (최소, 최대크기등의 속성) {적용할 CSS 구문}
    보통은 pc일경우 1280 이상으로 지정하는듯. 절대적이진 않음.
    and로 보조 속성을 계속 넣어줄 수 있다.
    이 상황에서 여러 디바이스에 공통적으로 들어가는 것들은 global 로 빼주는 형식으로 한다. > 코드 다이어트

    기기에 따른 @media 를 구성할때는 반복적인 캔슬을 방지하기 위해서 크기범위를 min-width, max-width 로 지정해주는것이 좋음.

    (orientation: landscape / portrait)
        화면을 어떤 상황에서 표시할지 지정하는 용도. 비율에 따른듯.
        landscape 는 가로가 더 긴화면, portrait은 세로가 더 긴화면.

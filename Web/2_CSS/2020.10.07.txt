
* CSS 에서 색상넣을때 rgb(번호, 번호, 번호) 도 가능. rgba(색상번호, 번호, 번호, 투명도 퍼센트) 를 넣으면 투명도 설정도 가능.

* 동일선상의 태그중에서도 우선순위를 정해줄 수 있는 G-index 라는게 있는듯?
* {background: url('이미지url');}
    배경에 이미지를 삽입. 배경에 가득 차도록 반복됨.
    background 는 뒤에 나오는것들을 축약한 축약법.
    url (이미지1) , url (이미지2) 해서 배경 이미지 2 개도 등록 가능. 거의 안쓰긴 하는데 쓸 때도 있음

* {background-size: cover / contain / 퍼센트 / 픽셀}
    배경의 크기를 지정할 수 있음.
    cover 는 해당 색으로 반복하지 않고 채움. 높이기준. 옆으로 넘어가버리는 이미지는 잘림.
    contain 은 해당 이미지로 가로 넓이로 채우는 느낌. 아래로 공간이 있으면 아래로는 반복된다. 종횡비.
    퍼센트는 가로 길이의 비율로 채움. 100% 는 contain 과 같은 결과인데 50%으로 하면 화면을 반으로 나눠서 이미지를 반복한다.
    값은 두개를 넣을 수 있는듯? 앞의 숫자가 witdh , 뒤의 숫자가 height 로 적용되는듯.
    픽셀은 해당 픽셀넓이로 이미지를 줄이는 기능. 원본크기가 다를경우 찌그러질수 있음.

* {background-repeat: repeat / repeat-x / repeat-y / no-repeat;}
    배경의 반복을 설정. 반복, x 축으로만 반복, y 축으로만 반복, 반복하지 않음.
    no-repeat 을 제일 많이 사용.

* {background-position: conter / top / bottom / }
    배경이미지의 출력을 시작할 시점...을 설정할 수 있다. 값은 x 축 과 y 축으로 구분해서 띄어쓰기로 구분해서 두개의 값을 같이 사용할 수 있음.
    픽셀로 구분하면 화변의 x축, y축의 해당시점에서 이미지를 시작. 축 기준점은 기본 왼쪽 위.
    축의 기준점 또한 속성으로 지정가능
    ex) right 10px top 20px > 오른쪽을 기준으로 10px 옆, 위를 기준으로 20px 아래?

* {background-attachment: scroll / fixed;}
    기본값이 scroll 이라서 자동완성으로 fixed 까지 나온다.
    스크롤에 관여받지 않고 화면의 지정된 위치에 고정된다.

* {background : color / url() / repeat / position / size / attachment}
    해당 순서로 적으면 모든 속성이 한번에 적용된다.

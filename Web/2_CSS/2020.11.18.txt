
* {visivility: visible / hidden / collapse;}
    visible 이 기본값. 변경점이 없음.
    hidden 숨김설정. 내용물을 보여주지 않을뿐 해당 태그가 가지고 있는 공간은 그대로 남아있는다.
    collapse 숨김설정. 하지만 내용물과 같이 공간도 함께 감춘다.
        * collapse 는 table 태그에만 적용된다.

* {text-shadow: a b c 색상;}
    텍스트에 그림자를 넣어주는 속성. 각각 수치를 넣는다.
    a = 그림자가 텍스트기준, 좌우로 얼마만큼이나 떨어지는지.
    b = 그림자가 텍스트기준, 상하로 얼마만큼이나 떨어지는지.
    c = 그림자의 블러효괴의 강도. 높을수록 글자의 번짐이 강해진다.

* {box-shadow: a b c 색상;}
    기본적인 속성값들은 text-shadow랑 같다.
    위에 있는 객체의 그림자는 아래 객제를 가려버리고 적용되는듯?
    우성순위가 객제1 > 객체1의 그림자 > 객체2 > 객체2의 그림자 같은 느낌.
    앞에 inset 속성값을 넣어주면 그림자가 안쪽으로 들어간다.

* {transition-duration: 시간s;}
    속성이 변화하는 시간을 지정할수 있다. 이걸로 간단한 애니메이션 호과를 줄 수 있음.
    변화하는건 오직 수치값 > 수치값 인걸 주의할것

* {transition-delay: 시간s;}
    속성의 변화를 시작하는데 걸리는 시간을 정해줄 수 있음.

* {transition-timing-function: ;}
    변화의 속도에 가속도 설정을 해줄 수 있음. 속성은 알아서 하기엔 너무 어렵고. 외부사이트나 구글 개발자툴을 이용해서 사용한다.

* {transition-property: 속성;}
    변화할 속성을 지정할 수 있음. 중복으로 지정하고 싶을땐 콤마로 구분해줌.

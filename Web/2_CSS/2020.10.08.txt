
* {list-style-image: url('');}
    목록의 표시를 이미지로 대체할 수 있다.

* {list-style-position: outside / inside }
    outside가 기본값.
    불릿(목록 표시)를 안쪽으로 넣을건지 바깥으로 뺄건지의 설정. 실무에서는 자주 안쓴다?

* {list-style : circle / disc / square ;}
    ul 목록에서 사용할 수 있는 것들.
    불릿 모양을 각각 속성명과 맟준애로 변경 가능. html 속성에서 했던애와 같음. 자주안씀.
* {list-style : decimal / lower-alpha / upper-alpha / lower-roman / upper-roman / none ;}
    ol 목록에서 사용할 수 있는 속성값들.
    10진수 숫자, 소문자 영어, 대문자 영어, 소문자 로마숫자, 대문자 로마숫자, 숫자 없음.

* 스프라이트 기법? 여러 이미지를 각각의 파일이 아니라 하나의 파일로 통합해서 가지고 있고 필요함에 따라 조건부로 특정 부분의 이미지를 불러오는 기법?

* background-position 으로 픽셀값 조정해서 스프라이트 기법을 사용할 수 있음. 기억해야할건 [x축값 y축값] 인데 왼쪽위에서부터 시작해서 지정된 값만큼 '오른쪽' '아래' 방향으로 밀어낸다는거. 오른쪽에 있는 이미지를 가져오려면 x축에 마이너스값을 넣어야함.

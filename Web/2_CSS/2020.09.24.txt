오늘은 CSS 시작.

* CSS 스타일을 넣는 방법은 크게 세가지
1. title 태그 아래에 link 태그로 외부 .css 파일을 불러온다.
2. title 태그 아래에 style 태그를 넣어 css를 직접 써넣는다.
3. 각각의 태그 안에 style 속성을 준다. 인라인 스타일 방식이라고 한다. 위치상 절대로 덮어씌워지지 않기 때문에 쓰지 않는다.
실무에서는 1번이 제일 많이 쓰임. 3번은 아예 안쓰임.

* 프로그램은 위에서 아래로 순차적으로 적용. 아래에 있는 코드가 위에 있는 코드를 덮어씌워서 적용된다. 

* .css 는 바로 태그잡아주고 스타일 넣어줘도 돼넹

* 확장자명 .css 도 쓴다. 실제로 작업할때는 이렇게 분리시켜놓고 나중에 가져오는방식을 사용한다. html 에서 .css 를 불러올때는 title 아래에 link 태그를 사용한다. 이걸 외부 스타일 방식이라고 한다.

* 프론트 취업에서는 CSS가 많이 중요하다.

* CSS 에서 id 를 불러올때는 # 을 붙이고 전체적인 태그를 불러올때는 그냥 태그이름만 써도 되고, 클래스를 불러올때는 앞에 . 을 붙인다. 여러개가 사용된 태그에서 특정 태그를 지정할때는 태그명[지정가능 속성]으로 넣어주면 인식하는듯.

* 화면분할 단축키 ctrl + alt + 오른쪽 방향키

* id 는 한 문서안에서 하나만 사용할 수 있지만, class 는 여러개를 사용할 수 있다. 심지어 한 태그에 여러개의 클래스를 중복해서 적용할 수도 있다. 할때는 class="1번 2번", 스페이스로 구분.

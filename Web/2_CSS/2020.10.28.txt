
* 태그::first-letter {;}
    태그에서 표현되는 글자의 첫 글자만 잡아내서 CSS 를 적용한다.
    위치이동이나 다른 CSS는 안들어가고 글자에 대한 속성만 적용된다.

* 태그::first-line {;}
    태그에서 표현되는 첫 줄에만 CSS 를 적용한다. 줄의 기준은 <br/> 태그인가? 
    얘도 위치이동이나 다른 CSS는 안들어가고 글자에 대한 속성만 적용된다.

* ::before , ::after
    지정된 태그의 앞이나 뒤에 공간을 만들어서 내용을 넣어주는거. float 캔슬러에서 사용하는거.
    태그 뒤에 새로운 공간을 만드는게 아니라 해당태그의 앞이나 뒤의 빈공간에 내용을 추가하는 개념? 태그 안에 끼워넣는느낌인듯.
    html 의 변경없이 태그처럼 활용할 수 있음.

* html Table 표에서 스프레드시트처럼 계산하려면 Java 적용해야 하는듯. C랑 구문자체는 비슷하긴 한데 데이터를 가져오는 선택자가 좀 헷갈리네. 적용하는것도 태그에 하나하나 아이디를 줘야하나?
